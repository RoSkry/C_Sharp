Конструкторы
/////////////////////////
  	Имя	Описание
	Stream	Выполняет инициализацию нового экземпляра класса Stream.

////////////////////////

Методы

////////////////////////
  	Имя	Описание
	BeginRead	Начинает операцию асинхронного чтения.
	BeginWrite	Начинает операцию асинхронной записи.
	Close	Закрывает текущий поток и отключает все ресурсы (например, сокеты и файловые дескрипторы), связанные с текущим потоком.
	CreateObjRef	Создает объект, который содержит всю необходимую информацию для создания прокси-сервера, используемого для взаимодействия с удаленным объектом. (Унаследовано от MarshalByRefObject.)
	CreateWaitHandle	Устаревшее. Размещает объект WaitHandle.
	Dispose	Перегружен. Освобождает все ресурсы, используемые объектом Stream.
	EndRead	Ожидает завершения отложенного асинхронного чтения.
	EndWrite	Заканчивает операцию асинхронной записи.
	Equals	Определяет, равен ли заданный объект Object текущему объекту Object. (Унаследовано от Object.)
	Finalize	Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора. (Унаследовано от Object.)
	Flush	При переопределении в производном классе очищает все буферы данного потока и вызывает запись данных буферов в базовое устройство.
	GetHashCode	Играет роль хэш-функции для определенного типа. (Унаследовано от Object.)
	GetLifetimeService	Извлекает объект обслуживания во время существования, который управляет политикой времени существования данного экземпляра. (Унаследовано от MarshalByRefObject.)
	GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
	InitializeLifetimeService	Возвращает объект обслуживания во время существования для управления политикой времени существования данного экземпляра. (Унаследовано от MarshalByRefObject.)
	MemberwiseClone	Перегружен. 
	Read	При переопределении в производном классе считывает последовательность байтов из текущего потока и перемещает позицию в потоке на число считанных байтов.
	ReadByte	Считывает байт из потока и перемещает позицию в потоке на один байт или возвращает -1, если достигнут конец потока.
	Seek	При переопределении в производном классе задает позицию в текущем потоке.
	SetLength	При переопределении в производном классе задает длину текущего потока.
	Synchronized	Создает потокобезопасную (синхронизированную) оболочку для заданного объекта Stream.
	ToString	Возвращает объект String, который представляет текущий объект Object. (Унаследовано от Object.)
	Write	При переопределении в производном классе записывает последовательность байтов в текущий поток и перемещает текущую позицию в нем вперед на число записанных байтов.
	WriteByte	Записывает байт в текущее положение в потоке и перемещает позицию в потоке вперед на один байт.

/////////////////////////////////////

Поля

///////////////////////////////////
  	Имя	Описание
	Null	Класс Stream без резервного хранилища.

///////////////////////////////////

Свойства

///////////////////////////////////

  	Имя	Описание
	CanRead	При переопределении в производном классе возвращает значение, показывающее, поддерживает ли текущий поток возможность чтения.
	CanSeek	При переопределении в производном классе возвращает значение, которое показывает, поддерживается ли в текущем потоке возможность поиска.
	CanTimeout	Получает значение, которое показывает, может ли для данного потока истечь время ожидания.
	CanWrite	При переопределении в производном классе возвращает значение, которое показывает, поддерживает ли текущий поток возможность записи.
	Length	При переопределении в производном классе получает длину потока в байтах.
	Position	При переопределении в производном классе получает или задает позицию в текущем потоке.
	ReadTimeout	Получает или задает значение в миллисекундах, определяющее период, в течение которого поток будет пытаться выполнить операцию чтения, прежде чем истечет время ожидания. 
	WriteTimeout	Получает или задает значение в миллисекундах, определяющее период, в течение которого поток будет пытаться выполнить операцию записи, прежде чем истечет время ожидания.
