Конструкторы
//////////////
  	Имя	Описание
	DriveInfo	Предоставляет доступ к сведениям на указанном диске.
///////////////


Методы

////////////
  	Имя	Описание
	Equals	Определяет, равен ли заданный объект Object текущему объекту Object. (Унаследовано от Object.)
	Finalize	Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора. (Унаследовано от Object.)
	GetDrives	Возвращает имена дисков всех логических дисков данного компьютера.
	GetHashCode	Играет роль хэш-функции для определенного типа. (Унаследовано от Object.)
	GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
	MemberwiseClone	Создает неполную копию текущего объекта Object. (Унаследовано от Object.)
	ToString	Возвращает имя диска в виде строки. (Переопределяет Object.ToString().)
///////////////////

войства
  	Имя	Описание
	AvailableFreeSpace	Указывает объем доступного свободного места на диске.
	DriveFormat	Получает имя файловой системы, например NTFS или FAT32.
	DriveType	Получает тип диска.
	IsReady	Получает значение, указывающее, готов ли диск.
	Name	Получает имя диска.
	RootDirectory	Получение корневого каталога диска.
	TotalFreeSpace	Получает общий объем свободного места, доступного на диске.
	TotalSize	Получает общий размер области хранения на диске.
	VolumeLabel	Получает или задает метку тома диска.
