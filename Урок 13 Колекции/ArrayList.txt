Конструкторы
	ArrayList - создается пустой список. 
	ArrayList ar = new ArrayList();
Как только 1-ый элемент добавляется в список его емкость увеличивается до 4, при добавлении 5-ого элемента – до 8 и т.д. При увеличении емкости список полностью копируется на новое место. 
	ArrayList(int capacity) задает начальную емкость списка
	ArrayList ar = new ArrayList(10);
	ArrayList(ICollection c) инициализирует список элементами коллекции


Емкость и количество элементов
	Свойство Capacity возвращает и задает емкость списка.
	Свойство Count возвращает количество элементов.
	TrimToSize() – устанавливает емкость коллекции равной ее текущему размеру.

Добавление элементов

	Add(object o) добавляет элемент в конец списка.
	AddRange(T[] o) добавляет диапазон элементов
	Insert(int pos, T o) вставляет элемент в указанную позицию
	InsertRange(int pos, T[] o)вставляет диапазон элементов

Доступ к элементам
	 по индексу или в цикле foreach

Удаление
	RemoveAt(int pos) удаление элемента в заданной позиции
	Remove(T o) удаление 1-ого элемента с заданным значением
	RemoveRange(int st, int col) удаление диапазона элементов, st – начальный индекс, col – количество
	RemoveAll(Predicate<T> cmp) удаляет все элементы, которые удовлетворяют предикату

Поиск элементов
	IndexOf(T o) поиск 1-ого элемента
	LastIndexOf(T o) поиск последнего элемента
	BinarySearch(T o) поиск в сортированном списке
	T Find(Predicate<T> cmp) ищет элемент, который удовлетворяет предикату
	List<T>  Find(Predicate<T> cmp) ищет все элементы, который удовлетворяют предикату


Сортировка
	Sort() – можно использовать, если элементы списка реализуют интерфейс IComparable
	Sort(IComparer) – передается объект, реализующий интерфейс для попарного сравнения элементов списка

Копирование элементов списка в массив
	CopyTo(Array a) копирует все элементы списка в массив, массив должен быть создан заранее и иметь размер достаточный для размещения всех элементов
	CopyTo(Array a, int st) копирует все элементы списка в массив, начиная с заданной позиции в массиве
	ToArray() создает новый массив object[] и копирует в него все элементы списка
	ToArray(Type t) создает новый массив заданного типа и копирует в него все элементы списка
