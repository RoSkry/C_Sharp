Stack - конструктор
////////////////////////////////////////
Имя	Описание
	Stack()	Инициализирует новый пустой экземпляр класса Stack с начальной емкостью по умолчанию.
	Stack(ICollection)	Инициализирует новый экземпляр класса Stack, который содержит элементы, скопированные из указанной коллекции, и обладает начальной емкостью, равной количеству скопированных элементов.
	Stack(Int32)	Инициализирует новый экземпляр класса Stack, который является пустым и обладает указанной начальной емкостью или емкостью по умолчанию, в зависимости от того, какое значение больше.

Методы
////////////////////////////////////////
  	Имя	Описание
	Clear	Удаляет все объекты из коллекции Stack.
	Clone	Создает неполную копию коллекции Stack.
	Contains	Определяет, входит ли элемент в состав коллекции Stack.
	CopyTo	Копирует элементы класса Stack в существующий одномерный массив класса Array, начиная с указанного индекса массива.
	Equals	Определяет, равен ли заданный объект Object текущему объекту Object. (Унаследовано от Object.)
	Finalize	Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора. (Унаследовано от Object.)
	GetEnumerator	Возвращает интерфейс IEnumerator для коллекции Stack.
	GetHashCode	Играет роль хэш-функции для определенного типа. (Унаследовано от Object.)
	GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
	MemberwiseClone	Создает неполную копию текущего объекта Object. (Унаследовано от Object.)
	Peek	Возвращает верхний объект стека Stack, но не удаляет его.
	Pop	Удаляет и возвращает верхний объект стека Stack.
	Push	Вставляет объект как верхний элемент стека Stack.
	Synchronized	Возвращает синхронизированную (потокобезопасную) оболочку коллекции Stack.
	ToArray	Копирует элементы Stack в новый массив.
	ToString	Возвращает объект String, который представляет текущий объект Object. (Унаследовано от Object.)


Свойства
////////////////////////////////////////
  	Имя	Описание
	Count	Получает число элементов, содержащихся в коллекции Stack.
	IsSynchronized	Получает значение, позволяющее определить, является ли доступ к коллекции Stack синхронизированным (потокобезопасным).
	SyncRoot	Получает объект, который можно использовать для синхронизации доступа к коллекции Stack.
